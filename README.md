# 轻松音乐 v1.1.2

基于electron-vue + vite 开发桌面端免费音乐软件替代方案
软件内所有内容均来源网络采集，请于下载后24小时内删除，

软件开发初衷是为了学习和研究，至停止维护之前不收费！

### 安装

```bash
$ npm install
```

### 运行

```bash
$ npm run dev
```

### 打包

```bash
# windows
$ npm run build:win

# macOS
$ npm run build:mac

# Linux
$ npm run build:linux
```
